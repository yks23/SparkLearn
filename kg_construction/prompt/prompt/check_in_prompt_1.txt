# Role
你是一个专注实体等价概念判断的学科专家，你擅长于判断指定实体是否存在于检索出的实体的信息中。

# Task
用户将会交给你一个指定实体，和一些被检索出的实体及其描述的集合，这些检索出的信息中可能存在或提及与指定实体等价的实体（名称稍有不同），请你进行判断，给出
指定实体是否存在于候选实体集合的判断
# Constraint
- 输出的内容要尽量简洁，不要冗余。
- 对于只要存在类似或者从属关系的实体，都可以认为存在。如果是名称描述简略或者完整而造成的区别，也认为存在。
- 如果描述的定语稍有不同，也认为相同
- 只要描述的是相近的实体就认为存在
- 如果检索出的信息中含有指定实体，也认为成功检索。比如如果提供的实体集合没有和指定实体一致的，但是它们的相关描述中提及了指定实体，也认为成功检索。
- 输出要以```json开始，以```结束，在这中间严格使用完整的json列表的格式。

# Output Template
```json
[
    true,
    false,
    ...,
    # true代表存在，false代表不存在
]
```

### 实体名称  \n尖端放电  \n\n### 实体信息  \n尖端放电是指由于导体尖端曲率较大，导致其表面电荷密度增大，从而在尖端附近产生强电场，引发电荷从尖端释放的现象。这种现象通常发生在高电压条件下，因为高电场强度能够使周围空气发生电离，形成导电通道，使得电荷能够通过这个通道放电。尖端放电不仅是一个重要的物理现象，而且在工程应用中有着广泛的影响。\n\n与避雷针的关系: 尖端放电的原理被应用于避雷针设计中，通过引导雷击电流安全地传导至地面，从而保护建筑物免受雷击损害。
### 实体名称  \n静电平衡  \n\n### 实体信息  \n静电平衡是指在导体内部，当电荷分布达到一种稳定状态时，电荷不再移动，电场强度在导体内部为零的状态。在这种状态下，导体上的电荷只分布在表面，并且电荷密度在表面不同位置可能不同，但整体上电荷的分布遵循一定的规律，如电荷在导体表面的尖端会更加密集。静电平衡的导体具有等势的特性，即导体表面的电势处处相等。\n\n与导体的关系: 静电平衡描述的是导体在静电场中达到的一种稳定状态，此时导体内部的自由电荷重新分布，直到内部电场为零。
### 实体名称  \n电场分布  \n\n### 实体信息  \n电场分布是指在特定边界条件下，空间中电场的强度和方向的具体配置。根据电磁学原理，在给定边界条件下，空间的电场分布和导体表面的电荷分布是唯一确定的。这意味着，一旦边界条件（如导体的形状、位置以及电势条件）确定，电场在空间中的分布也就确定了。电场分布是电磁场理论中的一个重要概念，它不仅描述了电场在空间中的状态，还与电荷分布、电势等物理量紧密相关。\n\n与唯一性定理的关系: 电场分布的唯一性是由静电场中的唯一性定理所确定的，这一定理保证了在相同的边界条件下，电场分布的解是唯一的。
